## Configure

```bash
export FLASK_APP=application.py
export FLASK_DEBUG=1
source DATABASE_URL
```

`DATABASE_URL`:

- 
  ```bash
  export DATABASE_URL='<heroku-database-uri>'
  ```

## Run App Locally

```bash
flask run
```
